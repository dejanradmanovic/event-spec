YAML Reference

namespace
Logical grouping, e.g. ecommerce
name
Snake-case identifier, e.g. product_viewed
event_name
Tracking name sent to analytics providers; defaults to name if omitted
display_name
Human-readable label shown in the UI
version
SchemaVer with hyphens: 1-0-0, 1-2-0, 2-0-0
status
draft · active · deprecated · deleted
type
track · page · identify · group · alias
Property types
string · number · integer · boolean · object · array
property_priority
event_wins · context_wins · merge — collision resolution when merging event props with context traits
sampling
Optional sampling block:
strategy: none · user_id_hash · random
rate: number between 0 and 1 (e.g. 0.1 = 10%)
required
true if the event must always be fired at its trigger point
destinations
Destination keys this event should be sent to, e.g. [segment, mixpanel]
context_properties
Context trait names merged into the event payload, e.g. [user_id, locale]
provider_overrides
Per-provider mappings keyed by provider name:
event_name: tracking name override for this provider
property_map: map of local property → provider property name
