## Why GPL v3?

GPL v3 was chosen to ensure that improvements and derivatives remain open — preventing bad actors from building proprietary products on top of this work without contributing back.

## Full license text

The complete license text is in the [`LICENSE`](https://github.com/dejanradmanovic/event-spec/blob/main/LICENSE) file at the root of the repository.

Key terms:

- **Copyleft** — anyone who distributes a modified version of event-spec must release their modifications under GPL v3.
- **Patent grant** — contributors grant users a license to any patents covering their contributions.
- **Tivoization protection** — users must be able to run modified versions on the hardware they received.
- **No additional restrictions** — distributors may not add restrictions beyond what GPL v3 already requires.
